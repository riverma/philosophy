## [1.0.0] - 2026-07-10

### Added
- First independent release of Philosophy Cards, extracted from `riverma.github.io` into its own repository and published at [philosophy.riverma.com](https://philosophy.riverma.com).
- Installable **Progressive Web App**: web manifest, app icons, and a cache-first service worker so the app runs fully offline.
- In-app footer with a privacy statement, version, and source/license link.
- AGPL-3.0 license.
